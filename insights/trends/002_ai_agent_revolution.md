## 1. AI 에이전트란?

### 1.1 정의

> **AI 에이전트 (AI Agent)**: 목표를 입력받으면 스스로 계획을 세우고, 도구를 사용하며, 결과를 검증하여 자율적으로 업무를 완수하는 AI 시스템.

**기존 AI와의 차이**:

| 구분 | 기존 AI (ChatGPT 등) | AI 에이전트 |
|-----|---------------------|------------|
| **상호작용** | 대화형 (1회성 질문-답변) | 자율형 (목표 → 완수) |
| **작업 범위** | 단일 응답 생성 | 다단계 복합 작업 |
| **도구 사용** | 제한적 (플러그인) | 무제한 (API, 브라우저, 파일) |
| **의사결정** | 인간 주도 | AI 자율 판단 |
| **예시** | "이메일 초안 작성해줘" | "경쟁사 가격 조사 → 엑셀 정리 → 이메일 발송" |

### 1.2 핵심 구성 요소

**1. 계획 수립 (Planning)**
- 목표를 단계별 작업으로 분해
- 예: "경쟁사 조사" → ① 웹 검색 → ② 데이터 추출 → ③ 스프레드시트 작성

**2. 도구 사용 (Tool Use)**
- API 호출: GitHub, Notion, Slack, Gmail 등
- 웹 스크래핑: Selenium, Playwright 자동 실행
- 파일 조작: 문서 읽기/쓰기, 이미지 편집

**3. 메모리 (Memory)**
- 단기 메모리: 현재 작업 맥락 유지
- 장기 메모리: 과거 작업 이력 학습

**4. 자기 검증 (Self-Reflection)**
- 결과 평가: "목표 달성했나?"
- 에러 처리: 실패 시 대안 시도

### 1.3 주요 플랫폼

| 플랫폼 | 특징 | 가격 | 사용 난이도 |
|-------|------|------|----------|
| **AutoGPT** | 오픈소스, 자율 실행 | 무료 | 높음 (개발자용) |
| **LangChain Agents** | Python 라이브러리, 커스터마이징 | 무료 | 중간 |
| **Zapier Central** | 노코드, 비즈니스 자동화 | $20-50/월 | 낮음 (비개발자용) |
| **Microsoft Copilot Studio** | 엔터프라이즈급, MS 통합 | $30/사용자 | 낮음 |
| **Google Project IDX** | 개발 에이전트, 클라우드 IDE | 무료 (베타) | 중간 |

## 4. 한계 및 리스크

### 4.1 기술적 한계

**1. 정확도 문제 (현재 70-85%)**
```
❌ 실패 사례:
- 복잡한 맥락 이해 부족
- 애매한 지시 오해석
- API 에러 시 무한 루프

⚠️ 대응책:
- 인간 검토 단계 필수 (Human-in-the-loop)
- 중요 작업은 자동화 제외
- 에러 알림 시스템 구축
```

**2. 도구 호환성**
- 모든 시스템에 API가 있는 것은 아님
- 레거시 시스템 통합 어려움
- 보안 정책으로 외부 API 차단

**3. 비용 폭증 리스크**
```
AutoGPT 무한 루프 사고 사례 (2025년 3월):
- 목표: "최고의 비즈니스 아이디어 찾기"
- Agent가 72시간 동안 멈추지 않고 실행
- OpenAI API 비용: $2,847 청구
- 실제 가치: 거의 없음

교훈: 실행 횟수 제한 필수 (예: 최대 20단계)
```

### 4.2 비즈니스 리스크

**1. 일자리 대체 우려**
- 단순 반복 업무 직군 감소
- 재교육 필요성 증가
- 조직 저항 관리 필요

**2. 책임 소재 불명확**
```
시나리오: Agent가 고객에게 잘못된 정보 전달
- 누구의 책임인가? (개발자? 운영자? AI 제공사?)
- 법적 리스크 (GDPR, 개인정보보호법)
- 브랜드 신뢰도 하락
```

**3. 과의존 위험**
- Agent 의존 → 직원 역량 저하
- Agent 장애 시 업무 마비
- 대체 불가능한 전문성 상실

### 가이드 3: 인간 검토 단계 필수

**Human-in-the-Loop 패턴**:

```python
# ❌ 나쁜 예: 완전 자동화
agent.execute("고객에게 환불 처리")  # 위험!

# ✅ 좋은 예: 인간 검토
result = agent.draft("환불 처리 초안 작성")
if human_approve(result):
    agent.execute(result)
else:
    human_manual_process()
```

**검토 수준 설정**:
| 업무 중요도 | 검토 수준 | 예시 |

### 가이드 6: 성과 측정 KPI

**3가지 핵심 지표**:

**1. 시간 절감률**
```
측정 방법:
  Before: 수동 작업 시간 (시간)
  After: 자동화 후 시간 (시간)
  절감률 = (Before - After) / Before × 100

목표: 첫 3개월 30% 이상
```

**2. 비용 절감액**
```
계산식:
  절감액 = 절감 시간 × 시급 × 근무일수

예시:
  월 100시간 절감 × $25/시간 = $2,500/월
  연간: $30,000
```

**3. 정확도 (Error Rate)**
```
측정 방법:
  정확도 = (성공 건수 / 전체 실행 건수) × 100

최소 기준: 80% 이상
목표: 95% 이상 (6개월 후)
```
