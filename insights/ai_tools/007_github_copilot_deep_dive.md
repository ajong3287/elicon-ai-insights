## 1. GitHub Copilot이란?

### 1.1 정의

**GitHub Copilot = IDE 안에서 작동하는 AI 코딩 어시스턴트**

OpenAI와 GitHub의 협업으로 탄생한 도구로, **수십억 줄의 공개 코드**를 학습한 AI가 개발자의 코딩을 실시간으로 도와줍니다.

**특징**:
- IDE 네이티브 플러그인 (별도 웹사이트 불필요)
- 타이핑하면서 바로 제안 (< 1초 응답)
- 주석으로 의도 전달 → 코드 자동 생성
- 여러 언어 지원 (Python, JS, TypeScript, Go, Ruby 등)
