## 1. Midjourney V6란?

### 1.1 정의
Midjourney V6는 **Discord 기반 AI 이미지 생성 플랫폼**입니다.

**핵심 특징**:
1. **프롬프트 기반**: 텍스트 설명 → 이미지 자동 생성
2. **상업적 품질**: 인쇄물, 광고, SNS 콘텐츠 사용 가능
3. **다양한 스타일**: 사실적 사진 ~ 일러스트 ~ 추상 예술
4. **반복 개선**: 같은 프롬프트로 여러 버전 생성 + 선택

- 예술적 스타일 (패션, 광고, 컨셉 아트)
- 사실적 인물 사진 (DALL-E 대비 30% 더 자연스러움)

- 브랜드 일관성 (같은 스타일 반복 생성)

- 텍스트 삽입 (로고, 타이포그래피)

**프롬프트 공식**:
```

2. 스타일 (Style): 사진 vs 일러스트 vs 유화 vs ...
3. 구도 (Composition): 클로즈업 vs 와이드 vs ...
4. 조명 (Lighting): 자연광 vs 스튜디오 vs 네온 vs ...
5. 분위기 (Mood): 밝은 vs 어두운 vs 신비로운 vs ...
6. 품질 (Quality): 8k, ultra detailed, photorealistic, ...
7. 파라미터: --ar, --v, --stylize, --chaos
```

### 팁 2: 파라미터 마스터 (창의성 조절)

**필수 파라미터**:

| 파라미터 | 설명 | 예시 |
|---------|------|------|
| `--ar` | 가로세로 비율 | `--ar 16:9` (유튜브 썸네일) |
| `--v` | 버전 선택 | `--v 6` (최신 버전) |
| `--stylize` | 예술성 (0~1000) | `--stylize 800` (매우 예술적) |
| `--chaos` | 창의성 (0~100) | `--chaos 50` (중간 랜덤) |
| `--sref` | 스타일 참조 | `--sref [URL]` (일관성 유지) |
| `--no` | 제외 요소 | `--no people` (사람 제외) |

**실전 조합**:
```
유튜브 썸네일: --ar 16:9 --v 6 --stylize 300
인스타그램 포스트: --ar 1:1 --v 6 --stylize 500
블로그 헤더: --ar 21:9 --v 6 --stylize 400
```

### 팁 3: Remix 모드 활용 (반복 개선)

**Remix 모드란?**:
- 생성된 이미지를 기반으로 프롬프트 수정
- 완전히 새로 생성하지 않고 일부만 변경

**활용 방법**:
```
1. 기본 이미지 생성
   "/imagine sunset over ocean"

2. 마음에 드는 것 선택 후 'Vary (Region)' 버튼

3. 특정 영역만 수정
   - 하늘 부분 선택 → "dramatic storm clouds" 추가
   - 바다 부분 유지

4. 최종 결과: 하늘만 바뀐 이미지
```

**효과**: 완전히 새로 생성하는 것보다 5배 빠름
