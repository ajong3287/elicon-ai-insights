## 1. Perplexity AI란?

### 1.1 정의

> "ChatGPT + Google 검색 = Perplexity AI"

Perplexity AI는 **대화형 AI 검색 엔진**으로, 질문에 대한 답변을 생성하면서 동시에 최신 웹 데이터를 실시간으로 검색하여 출처를 명시합니다.

**기존 도구와 비교**:
| 도구 | 장점 | 단점 | Perplexity 차이점 |
|------|------|------|-------------------|
| **Google** | 방대한 데이터 | 수동 탐색 필요 | 자동 요약 + 재질문 |
| **ChatGPT** | 자연어 대화 | 2021년 데이터 (무료) | 실시간 웹 검색 |
| **Bing AI** | MS 통합 | 느린 속도 | 빠른 응답 (2-5초) |

### 1.2 핵심 기능

**1. 실시간 웹 검색**
- 최신 뉴스, 통계, 가격 정보 즉시 반영
- 2026년 1월 데이터까지 실시간 접근

**2. 출처 자동 인용**
- 모든 답변에 출처 링크 [1][2][3] 표시
- 클릭 한 번으로 원문 확인 가능

**3. 대화형 조사**
- "더 자세히 설명해줘" → 자동 심화 조사
- 이전 대화 맥락 유지 (세션당 100+ 메시지)

**4. 다중 소스 종합**
- 한 번에 10-20개 웹사이트 분석
- 상충되는 정보 자동 비교

**5. 이미지/동영상 검색**
- Pro: 이미지 업로드 + 분석
- YouTube, 논문 직접 검색

## 2. 실전 활용 시나리오 TOP 5

### 시나리오 1: 경쟁사 조사 (90% 시간 절약)

**Before (Google)**:
```
1. "Notion 경쟁사" 검색 → 10개 사이트 클릭
2. 각 사이트에서 정보 수동 복사
3. 엑셀에 정리 (30분)
```

**After (Perplexity)**:
```
질문: "Notion의 주요 경쟁사 5개와 각각의 강점, 가격, 시장 점유율을 표로 정리해줘"
결과: 3분 내 출처 포함 표 제공 ✅
```

**시간 절감**: 30분 → 3분 (90% ↓)
**ROI**: 월 10회 조사 시 4.5시간 절약 = $180 가치

**질문**: "개인정보보호법 개정안 2025년 12월, 주요 변경 사항과 스타트업 영향은?"

**전문가 검증 필요**: 법률 조언은 변호사 확인 필수 ⚠️
